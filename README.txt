# Travel Recommendation Website

## Project Overview
This travel recommendation website helps users discover ideal destinations based on their preferences. It features interactive displays, detailed descriptions, and multimedia content to enhance the user experience.

## Features
- **Home Page**: A welcoming introduction and recommendations for travel destinations.
- **About Us Page**: Information about the team and the mission of the website.
- **Contact Us Page**: A form for users to submit inquiries or feedback.
- **Personalized Recommendations**: Suggestions for beach, temple, and country destinations with accompanying images.
